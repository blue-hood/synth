# web-synthesizer

## Development

```
$ docker-compose up -d
$ docker-compose exec dev yarn
$ docker-compose exec dev yarn dev
```

## Linting and Formatting

```
$ docker-compose up -d
$ docker-compose exec dev yarn
$ docker-compose exec dev yarn fix
```

## Test

```
$ docker-compose up -d
$ docker-compose exec dev yarn
$ docker-compose exec dev yarn test
```
